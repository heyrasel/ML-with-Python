Inheritance:

Ekta class er jinishpotro onno class theke access kora.

Multiple Inheritance:

Multi-level inheritance:
ekta class er access arekta class e and arekta class er access arekta class e.

Hierarchical Inheritance:
Jokhon ekta class er access eker odhik class ke dibo tokhn eta Hierarchical inheritance hobe.

Encapsulation dara data accessability ke control kora jai like private,public,default etc diye.
